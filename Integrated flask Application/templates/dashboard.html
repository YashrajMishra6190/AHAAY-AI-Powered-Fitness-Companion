AHAAY Tracker

The AHAAY Tracker is a real-time fitness tracking tool that uses MediaPipe Pose Detection to monitor body movements through a webcam. It counts exercise reps, gives posture feedback, and displays performance using a Streamlit-based interface, helping users improve form and track progress without any wearable devices.

AHAAY Chatbot

The AHAAY Chatbot is an AI-powered virtual health assistant that provides personalized meal, fitness, and wellness recommendations based on the user’s BMI, gender, and health goals. It uses a Transformer model to generate real-time, interactive responses and is integrated into the web dashboard for easy access.
